# Sesión 15: Documentación, demostración y sustentación

## Objetivo

Preparar la entrega final mediante documentación breve, evidencia de pruebas y una demostración técnica clara del sistema construido.

## Contenidos

- Estructura de README o memoria técnica.
- Capturas o evidencia funcional.
- Explicación del modelo, arquitectura y persistencia.
- Preparación de la demostración del proyecto.

## Práctica guiada

- Documentar dependencias, estructura y modo de ejecución.
- Registrar casos de prueba y resultados.
- Ensayar la presentación del flujo principal.

## Reto de sesión

Explicar decisiones de diseño con lenguaje técnico claro y sin depender del código fuente como guion.

## Entregable mínimo

Proyecto documentado, con guía de ejecución, evidencia funcional y preparación para sustentación.
