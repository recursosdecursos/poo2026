# Programación Orientada a Objetos 2026-2

Este libro digital organiza el curso de POO alrededor de un producto integrador: un sistema de ventas de escritorio desarrollado en Java con IntelliJ IDEA, JavaFX, JDBC y SQLite.

## Propósito del curso

El estudiante diseña e implementa una solución orientada a objetos que evoluciona desde un modelo de dominio en memoria hasta una aplicación de escritorio con persistencia y documentación técnica.

## Herramientas de trabajo

- Java como lenguaje principal.
- IntelliJ IDEA como entorno de desarrollo.
- Maven para dependencias y compilación.
- JavaFX para interfaz gráfica.
- JDBC y SQLite para persistencia relacional.

## Progresión del curso

| Unidad | Sesiones | Resultado esperado |
|---|---:|---|
| U1. Fundamentos de POO | 1-6 | Modelo del dominio funcional en memoria. |
| U2. GUI y persistencia | 7-12 | Aplicación de escritorio con CRUD y base de datos. |
| U3. Proyecto integrador | 13-16 | Sistema completo documentado y sustentado. |

## Cómo usar este libro

Cada sesión presenta el objetivo, los contenidos, la práctica guiada y el entregable mínimo. El flujo sugerido es:

```text
Comprender el caso -> modelar -> implementar -> probar -> refinar -> explicar
```

## Navegación sugerida

1. Revise el [sílabo](Silabo_POO.md).
